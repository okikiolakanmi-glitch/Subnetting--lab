# Subnetting Lab

## Objective
The objective of this project is to understand subnetting, CIDR notation, subnet masks, and host calculations in networking.
[subnetting-cidr-calculations.jpeg] 
## Topics Covered
- Class A, B, C IP addresses
- Subnet masks
- CIDR notation
- Network address
- Broadcast address
- Number of hosts per subnet

## Example Subnet Calculation
IP Address: 192.168.1.0/24  
Subnet Mask: 255.255.255.0  
Number of Hosts: 254  
Network Address: 192.168.1.0  
Broadcast Address: 192.168.1.255  

## Tools Used
- Manual binary calculation
- Subnet calculator

## Conclusion
This project helped me understand how subnetting works and how to calculate subnet masks, network addresses, and number of hosts.
